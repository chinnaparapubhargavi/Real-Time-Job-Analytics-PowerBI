# Real-Time Job Analytics Portal (Power BI)

This project was completed as part of the ElevanceSkills Internship Program.

## Tools Used
- Power BI
- Excel / CSV Dataset

## Project Overview
This project analyzes real-time job market data using Power BI.  
It focuses on understanding job roles, company distribution, gender preferences,
qualifications, company size, geographic filters, and time-based conditions.

## Key Features
- Role-based filtering (Data Engineer, Data Scientist, etc.)
- Country and region exclusions
- Gender and qualification-based analysis
- Company size filtering
- Time-based conditions applied as per task requirements
- Interactive dashboards with slicers and filters

## Visualizations
- Bar charts
- Scatter plots
- Tree maps
- Interactive Power BI dashboards

## Deliverables
- Power BI Dashboard
- Project Report (PDF)
